# Discord Bot

## Requirements
- discord.js
- tsnode
- nodemon
```
npm i -g tsnode nodemon
npm i discord.js
```
</br>

## Installation
```
npm install
```
<span style="color: red;">Dont forget to edit the `.env` file to include your bot token.</font>
</br></br>


## Running App in Developer mode
```
npm run dev
```
